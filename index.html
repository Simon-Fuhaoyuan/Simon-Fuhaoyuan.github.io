Selected Projects

Optimizing Caffe

Course Project (CS433, SJTU)
Nov. 2019 - Jan. 2020
Link:
Caffe is an awesome structure producing its own system of deep learning, getting rather high performance with the help of CUDA kernel functions, cudnn and cublas library. Our task here is to replace cublas functions in Caffe source code with our own CUDA kernel functions to accelerate ResNet101 and Inception21k network, respectively. Since cublas library functions do matrix calculation regardless with its shape, while our customized kernel functions can optimize matrix calculation case by case when given the CNN architecture, which makes it possible for acceleration. However, cublas is a hardware-level optimization for matrix calculation. We have tried tricks like data tiling, shared memory, loop unrolling, stream parallel, constant memory, kernel fusion, cache friendly operation, etc, and we gained great acceleration compared with Caffe with only general kernel functions, but we still cannot defeat cublas.

Image Classification and Segmentation

Course Project (CS385, SJTU)
May. 2020 - Jul. 2020
Link:
This part includes two course projects for Machine Learning, CS385. In the first project, I implemented mutiple machine learning algorithms from traditional ones like logistic regression, kernel regression, LDA, SVM to modern ones like MLP, CNN (VGG11, ResNet18) to work on classification tasks of MNIST dataset. I also tried GAN to generate some figures and used GradCAM on trained CNNs to get heat map with interpretability. In the second project, we worked in group to finish three tasks: multi-label classification, segmentation and VAE for generation, where I was responsible for image segementation using Pascal VOC 2012 dataset and implemented FCN and U-Net using pytorch. In conclusion, these two projects are not aimed to design new network architecture or new algorithms, but just focus on understanding theory of machine learning.
